# 🛡️ Security Operations Center (SOC)

Корпоративная панель управления безопасностью для службы безопасности бизнеса.

**Полностью автономное решение** — все данные хранятся локально в браузере, никаких внешних зависимостей.

## 📦 Модули системы

### 📊 Панель инцидентов (`dashboard.html`)
- Создание, отслеживание и закрытие инцидентов ИБ
- Счётчики: критические / в работе / закрытые
- Смена статусов в один клик
- Локальное хранилище с возможностью миграции на бэкенд

### 📹 Видеонаблюдение (`cameras.html`)
- 4 камеры с индикацией статуса (онлайн/офлайн)
- Полноэкранный режим
- Снимки с камер
- Поддержка WebRTC для подключения реальных камер
- Журнал событий видеонаблюдения

### 🔐 Контроль доступа СКУД (`access-log.html`)
- Журнал проходов сотрудников
- Фильтрация по дате, зоне, статусу, карте
- Цветовые индикаторы: разрешён / отклонён / тревога
- Экспорт в CSV

### 🚨 Оповещения (`alerts.html`)
- Три уровня тревог: критические / предупреждения / информационные
- Управление статусами оповещений
- Звуковое оповещение при критических событиях
